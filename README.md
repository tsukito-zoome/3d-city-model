3D City Model from Aerial Point Cloud Data

大学の演習授業（画像・映像コンテンツ演習）にて、4人チームで制作した3D都市モデル生成プロジェクトです。
東京駅〜後楽園キャンパス周辺を対象に、航空測量で取得した点群データを処理し、PCやVRシステムで閲覧可能な3Dコンテンツを生成します。

概要

飛行機による上空測量で取得した1m間隔の高さデータ（地面・建物・樹木・車などが混在）を処理し、視覚的に美しい3D都市モデルを生成します。

担当・独自アイデア：直方体化（bokuseru.py）

過去の履修学生が誰も実施していなかった独自アプローチとして「直方体化」を提案・実装しました。


建物をクラスタリングで分類し、各建物を覆う最小の直方体で表現
直方体内部の点をすべて削除することで大幅なデータ削減を実現
都心部のビルが多いエリアの特性を活かし、高層建物に絞って適用
デモ＆ポスター発表にて、外部評価者（教授・院生）から独自アイデアとして高く評価

使用技術
